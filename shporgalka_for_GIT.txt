
Конфігурація
git config --global user.name «[name]» - встановити ім'я, яке буде прикріплюватися до коміту.
git config --global user.email «[email address]» - встановити email, який буде прикріплюватися до коміту.


Створення репозиторіїв
git init [project-name] - створити новий локальний репозиторій із заданим ім'ям.
git clone [url] - завантажити проєкт і його повну історію змін.


Робота з файлами
git rm [file] - видалити файл із робочої директорії та додати в індекс інформацію про видалення.
git rm --cached [file] - видалити файл із репозиторію, але зберегти його локально.
